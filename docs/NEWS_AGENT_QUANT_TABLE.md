# 新闻 Agent 量化表设计

本项目输出 A 股研究辅助型操作建议，不自动交易，不接券商接口，不承诺收益。新闻结论是买入/卖出/加减仓/持有/等待建议的重要输入之一，但新闻通道本身不得单独触发操作，必须和量价、财报、同行、BookSkill、筹码和反证共同确认。

## 目标

新闻通道不是简单判断“利好/利空”。它的目标是把外部信息拆成可比较、可复核、可回测的结构化表，帮助回答三个问题：

- 这家公司自身有没有重要事件？
- 同行、政策、地域背景是否已经变化？
- 它相对同行是被更多关注、被更少关注，还是同行有事但它暂时沉默？

这正是 agent-based model 相对普通机器学习特征表的优势：agent 可以先理解事件层级和信息来源，再把判断压缩成可解释的数值字段。

## 信息分层

| 层级 | 说明 | 当前落地状态 | 典型字段 |
|---|---|---|---|
| 股票自身新闻 | 公司公告、订单、中标、业绩、处罚、减持、诉讼、产品技术 | 已接入 | `news_self_intensity_score_30d`、`news_warning_score_30d`、`news_opportunity_alert_score_30d` |
| 领域/同行背景 | 同行业股票是否集中出现新闻、风险或机会 | 已接入基础版 | `peer_group_news_count_avg`、`peer_group_news_risk_avg`、`news_peer_attention_gap_30d` |
| 政策/宏观背景 | 政策、价格、补贴、关税、利率、汇率、宏观市场新闻 | 已接入标题关键词版 | `news_policy_background_score_30d` |
| 地域背景 | 地方政策、区域灾害、区域产业链、地方监管 | 预留字段，未编造 | `news_region_background_score_30d` |
| 证据质量 | 官方公告、公开聚合新闻、人工复核、模型推断 | 已接入基础版 | `news_evidence_quality_score_30d` |

## 相对值很重要

绝对新闻量容易误导。大公司天然新闻更多，热门行业天然新闻更多，所以必须保留相对字段。

| 相对关系 | 判断含义 | 字段 |
|---|---|---|
| 自身新闻量高于同组平均 | 可能存在关注度升温，需结合趋势和事件类型确认 | `news_peer_attention_gap_30d`、`news_relative_attention_score_30d` |
| 同行被频繁提及但自身沉默 | 可能是信息滞后，也可能是公司与行业主线脱节 | `news_peer_silent_while_group_active_30d` |
| 同行风险升高 | 个股自身无风险新闻也要谨慎复核 | `peer_group_news_risk_avg` |
| 政策/价格背景被提及 | 不直接定性利好利空，需要结合产业链位置 | `news_policy_background_score_30d` |

## 当前新闻预警公式

风险预警分：

```text
news_warning_score_30d =
  0.45 * 明确风险事件
+ 0.25 * 正负冲突强度
+ 0.08 * 新闻关注度异常
+ 0.10 * 同组新闻风险
+ 0.15 * 近期负面净重大性
+ 0.15 * 同组活跃但自身沉默
```

机会提醒分：

```text
news_opportunity_alert_score_30d =
  0.28 * 订单/中标/业绩/技术等机会事件
+ 0.30 * 新闻关注度异常
+ 0.12 * 自身曝光高于同组平均
+ 0.05 * 政策/宏观/价格背景
+ 0.05 * 证据质量
- 0.25 * 风险预警分
```

解释规则：

- `风险预警`：`news_warning_score_30d >= 1.0`，需要人工复核风险来源。
- `机会提醒`：机会提醒分大于 0 且风险预警未触发，说明值得看，不等于正面结论。
- `无明显新闻提醒`：新闻层不提供增量证据，不能因此认为公司没有风险。

## 回测使用边界

- 新闻必须满足 `available_at <= decision_date 15:00`，不得使用未来新闻。
- 只有日期没有具体时间的公告，保守按下一自然日可用处理。
- 免费公开聚合新闻不等于交易所/巨潮官方原文，重大事件必须人工复核。
- 新闻机会单独使用容易过拟合，必须和环境 Gate、长期趋势、Book Skill、反证一起使用。
- 地域背景没有可信字段时只输出 0 或信息不足，不得由模型猜测地域事件。

## 用户怎么看

推荐给用户展示四句话：

1. 自身是否有重大新闻或公告。
2. 同行/政策/地域背景是否有变化。
3. 自身相对同行是升温、沉默还是同步。
4. 新闻层支持的是“继续深挖、放入观察、暂时剔除、信息不足”中的哪一种研究分级。

新闻层永远不单独输出投资动作。
