# Book Skill 来源优先级

## 核心优先级

| 优先级 | 书籍 | 用途 |
|---:|---|---|
| 1 | 专业投机原理 | 趋势、风险、反转、投机纪律、资金管理 |
| 1 | 日本蜡烛图技术 | K线形态、反转/持续、支撑阻力、确认 |
| 1 | 道氏理论 | 市场状态、趋势级别、成交量确认、指数确认 |
| 2 | 海龟交易法则 | 突破、波动率、轻量回测、仓位风险 |
| 2 | 股票作手回忆录 | 市场心理、等待、错误复盘、消息陷阱 |
| 3 | 安全边际 | 估值、安全边际、下行风险 |
| 3 | 手把手教你读财报 | 财务排雷、现金流、资产质量 |

## 调用规则

涉及趋势/支撑压力/高抛低吸/买卖观察区时：

1. 必须优先检索三本核心书。
2. 再调用《海龟交易法则》和《股票作手回忆录》做风险和纪律补充。
3. 最后用财务和估值类书籍做反证。

涉及财务排雷时：

1. 优先《手把手教你读财报》。
2. 再用《安全边际》做估值和下行风险约束。

涉及估值时：

1. 优先《安全边际》。
2. 再用趋势类书籍判断价格行为是否已经背离基本面证据。
